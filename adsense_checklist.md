# 🔮 Facetiny 구글 애드센스(Google AdSense) 승인 가이드 & 체크리스트

보내주신 애드센스 거절 안내 문서(`adsense_help.md`)의 모든 참고 링크 및 구글 정책 기준을 분석하여, 페이스타이니(Facetiny)가 애드센스 승인을 받기 위한 행동 지침과 기술 체크리스트를 정립했습니다.

---

## 📋 1. 구글 애드센스 주요 정책 요약 & 대응 현황

구글 애드센스 심사 통과를 위해 요구되는 핵심 기준과 현재 Facetiny의 반영 상태입니다.

### ① 최소 콘텐츠 요건 & 빈약한 콘텐츠 (Thin Content) 예방
*   **구글 기준**: 독창적이고 전문적이며 사용자에게 가치를 주는 장문의 글(최소 1,000~1,500자 이상)이 충분히 축적되어 있어야 합니다. 단순히 복사해서 붙여넣은 글이나 부실한 페이지만 있는 사이트는 거절됩니다.
*   **대응 완료**: 동양 전통 상학 고전인 《마의상법》과 현대 안면 계측 기하학을 융합한 전문 관상학 칼럼 **15개 편(평균 1,800자 이상)**을 직접 고유 문장으로 집필 및 구축 완료했습니다.

### ② 게시자 콘텐츠가 없는 화면에 광고 게재 금지 (준비 중인 화면)
*   **구글 기준**: 로봇이 사이트 첫 주소(`facetiny.pages.dev/`)에 들어왔을 때, React SPA(싱글 페이지 앱)의 경우 자바스크립트 실행 지연으로 인해 빈 템플릿(Loading 화면)만 보여 "콘텐츠가 없는 사이트"로 오인하고 승인을 반려하는 경우가 빈번합니다.
*   **대응 완료**: 
    *   15개의 모든 칼럼을 각각 독립된 정적 웹페이지(`public/blog/art_X/index.html`)로 자동 빌드하도록 시스템을 보강했습니다.
    *   메인 `index.html` 하단에 검색 로봇만 볼 수 있는 **숨겨진 사이트맵 링크 리스트**를 주입했습니다. 구글봇이 메인 홈만 스캔해도 15개의 유익한 장문 칼럼 주소들을 즉시 추적할 수 있습니다.

### ③ 탐색 및 이동의 편의성 (우수한 사용자 환경)
*   **구글 기준**: 사용자가 사이트 내에서 길을 잃지 않고 다른 콘텐츠로 유기적으로 이동할 수 있어야 합니다.
*   **대응 완료**: 
    *   모든 블로그 페이지의 최하단에 서로 다른 칼럼으로 자연스럽게 순환하는 **내부 링크 고리(CTA)**를 톱니바퀴 형태로 배치하여 체류시간을 극대화했습니다.
    *   정적 페이지마다 상단에 메인 서비스로 복귀하는 "실시간 무료 스캔 시작" 버튼을 배치해 유연한 유저 플로우를 확보했습니다.

## 🛠️ 2. 애드센스 재검토 요청 전 행동 체크리스트

코드를 배포한 후, 구글 애드센스 콘솔에서 **"검토 요청"**을 누르기 전에 아래 항목들을 순서대로 수행해 주세요.

### 1단계: 구글 서치 콘솔(Google Search Console) 등록 (필수 ⭐)
구글 봇에게 정적 블로그 주소와 사이트맵이 존재함을 알리는 작업입니다.
1.  [구글 서치 콘솔](https://search.google.com/search-console)에 접속하여 `https://facetiny.pages.dev/` 속성을 등록합니다.
2.  **Sitemaps** 메뉴로 이동하여 새 사이트맵 추가 창에 **`sitemap.xml`**을 입력하고 [제출]을 클릭합니다.
    *   *(우리가 생성한 `https://facetiny.pages.dev/sitemap.xml` 지도가 구글에 등록되어 봇이 하루 이내에 모든 글을 수집해 갑니다.)*

### 2단계: 광고 차단 프로그램(AdBlocker) 비활성화
*   애드센스 대시보드가 정상적으로 작동하지 않거나 사이트 연동 코드가 로드되지 않는 현상을 방지하기 위해, 크롬 등 브라우저의 광고 차단 확장 프로그램(AdBlock, uBlock 등)을 완전히 중지하고 페이지를 새로고침해 주세요.

### 3단계: 로컬 테스트 및 정적 배포 확인
*   개발용 서버를 켜거나 Cloudflare Pages 배포 후 아래 경로들이 정상적으로 열리는지 주소창에 직접 입력하여 테스트합니다:
    *   `https://facetiny.pages.dev/robots.txt` (텍스트 파일 정상 출력 확인)
    *   `https://facetiny.pages.dev/sitemap.xml` (사이트맵 구조 정상 확인)
    *   `https://facetiny.pages.dev/blog/art_1/` (정적 페이지 렌더링 및 하단 추천 링크 클릭 작동 확인)
